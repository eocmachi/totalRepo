    기본적인 박스의 성질에 대한 이해 width height padding margin

    1. 박스는 기본적으로 width와 height가 auto이다.

    2. 박스는 기본적으로 투명하고 y축으로 쌓여간다.

  • 기본 박스 02

    선택자에 대한 이해

    1. 모든 선택자는 *

    2. :nth-child(N)= 부모안에 모든 요소 중 N번째 요소
      A :nth-of-type(N)= 부모안에 A라는 요소 중 N번째 요소

    3. # id는 . class 보다 상위 선택자이다
      id > class > 요소

    4. 인접 형제 선택자, 일반 형제 선택자

      - 인접 형제 선택자의 문법 형태는 A + B
      A 바로 뒤에 오는 B만 선택된다

      - 일반 형제 선택자의 문법 형태는 A ~ B
      A 뒤에 오는 모든 B들이 선택된다
