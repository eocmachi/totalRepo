기본 박스 : box-model

  • 기본 박스 01

    기본적인 박스의 성질에 대한 이해 width height padding margin

    1. 박스는 기본적으로 width와 height가 auto이다.

    2. 박스는 기본적으로 투명하고 y축으로 쌓여간다.

    3. 박스는 기본적으로 block 옆에 block이 오지 못하게 한다.


  • 기본 박스 02

    선택자에 대한 이해

    1. 모든 선택자는 *

    2. :nth-child(N)= 부모안에 모든 요소 중 N번째 요소
      A :nth-of-type(N)= 부모안에 A라는 요소 중 N번째 요소

    3. # id는 . class 보다 상위 선택자이다
      id > class > 요소

    4. 인접 형제 선택자, 일반 형제 선택자

      - 인접 형제 선택자의 문법 형태는 A + B
      A 바로 뒤에 오는 B만 선택된다

      - 일반 형제 선택자의 문법 형태는 A ~ B
      A 뒤에 오는 모든 B들이 선택된다

Display & Position

  • Display 01

    1. inline 요소는 w,h, 상하m,p 값을 줄 수가 없다
    2. inline-block요소의 부모에 text-align:center로 박스를 가로로 배열할 수 있다
    3. inline-block 요소 사이의 간격을 붙이려면 font-size:0; 을 해주어야 한다
    4. inline 요소를 inline-block요소로 바꾸면 박스 사이의 간격이 4px 정도 생긴다


  • Position 01

    1. relative - absolute 관계에 대해서 배움
    2. fixed와 sticky 둘의 차이에 대해서도 배웠다

  • Position 02

    1. position: sticky; 에 대해 배움
    sticky는 요소가 컨테이너보다 커서 스크롤이 생겨날 때, 스크롤이 되거나 말거나 자기 상위 요소에 달라붙는다.
    끈끈하게 달라붙는다.

  • Position 03

    1. filter: grayscale( ) brightness( ); 를 새롭게 배움
    2. position: sticky;을 사용해서 웹사이트 만들어 봄
